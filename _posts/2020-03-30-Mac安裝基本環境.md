>製作者:Ericsiang

###### tags: `Mac` `base envirment`

## 先安裝 command line tools
```
xcode-select --install 
```
   
## 接著安裝 Homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```


**有了Homebrew  一切好辦事    
所以一定要安裝  
不過安裝Homebrew之前 
要先裝xode command line tools**

<br>

## Homebrew 安裝 git
```
brew install git
```
